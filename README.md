🛍️ FashionShop

Proyecto final del ciclo formativo Desarrollo de Aplicaciones Multiplataforma (DAM).
Aplicación móvil de tienda de ropa desarrollada con React Native, Node.js y MySQL.

📱 Descripción

FashionShop es una aplicación móvil que simula el funcionamiento de una tienda online de ropa.
La aplicación permite a los usuarios registrarse, iniciar sesión, ver productos, añadirlos al carrito y realizar un pago simulado.

El sistema está conectado a una base de datos MySQL mediante un servidor desarrollado con Node.js y Express, siguiendo una arquitectura cliente-servidor similar a la utilizada en aplicaciones reales.

También se ha implementado un sistema de administrador que permite añadir y eliminar productos desde la propia aplicación.

Este proyecto se ha desarrollado como trabajo final del ciclo formativo DAM, con el objetivo de aplicar los conocimientos de programación, bases de datos y desarrollo de aplicaciones móviles.

🧠 Arquitectura del proyecto

El proyecto sigue una arquitectura cliente-servidor:

📱 App móvil → React Native

🌐 API → Node.js + Express

🗄️ Base de datos → MySQL

La aplicación se comunica con el servidor mediante una API REST en formato JSON.
