# 🛍️ FashionShop

Aplicación de tienda de ropa desarrollada como proyecto final.

## 📱 Descripción
FashionShop es una aplicación móvil donde los usuarios pueden ver productos de ropa, registrarse, iniciar sesión y realizar compras.

## 🚀 Tecnologías utilizadas

### Backend
- Node.js
- Express
- MySQL

### App móvil
- Android Studio (Java/Kotlin)

## ✅ Funcionalidades actuales

✔ API funcionando  
✔ Conexión a MySQL  
✔ Endpoint para obtener productos  
✔ Productos cargados en la base de datos  

## 🔜 Próximas funcionalidades

- Login seguro
- Panel administrador
- Carritos de compra
- Sistema de pedidos
